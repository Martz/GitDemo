# Linux Commands

These are all generally useful day to day

 * ls - list files and folders
 * cd foo - change into the foo folder
 * pwd - present working directory
 * cd.. goes down a directory
 * ls -l additional file details
 * touch creates a file eg touch a-file.md
 * Learn vim!
 * cat shows whats in the file
 * clear cleans the console
 * cat FILE.md | less
 * cat FILE.md | wc -l is a word count
 * cd .. home directory 
 * tab twice will show whats in the directory
 * cd always goes to home directory 
 * . will act in the current directory eg code . 
 * du disk usage 
 * du -h human readable format 
 du -c -h -d1. list each folder with sizes in human readable format
